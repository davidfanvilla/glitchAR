# Retrato WebAR

Experiencia WebAR con marcador Hiro, Three.js/A-Frame, AR.js y MediaPipe Hands.

## Como usarlo

1. Abre la pagina del proyecto en Safari del iPhone.
2. Permite la camara.
3. Abre `webar-marker.html` en otra pantalla o imprimelo.
4. Apunta la camara del iPhone al marcador Hiro.
5. Cierra una mano para arrugar el retrato, abre la mano para desplegarlo y usa dos manos para girarlo en 3 ejes.

## Nota

Esta version no necesita Xcode. Usa WebAR con marcador para poder funcionar desde Safari del iPhone.
